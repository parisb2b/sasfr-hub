5. ÉTAPE 4 — DNS (À FAIRE MANUELLEMENT PAR L'UTILISATEUR)
================================================================================

Pour faire pointer le domaine racine sasfr.com vers ce déploiement :

1. VÉRIFIER D'ABORD qu'aucun enregistrement A/AAAA existant sur la racine
   de sasfr.com n'entre en conflit (attention aux enregistrements IONOS résiduels
   comme ceux rencontrés sur listes.sasfr.com).

2. Configurer chez OVH l'enregistrement pour le domaine APEX (racine) :
   - Un CNAME standard ne fonctionne PAS sur un domaine racine (apex).
   - Option A : ALIAS/ANAME vers cname.vercel-dns.com (si OVH le supporte)
   - Option B : Ajouter le domaine dans Vercel → Settings → Domains,
     Vercel fournira une adresse IP A à utiliser dans la zone DNS OVH

3. Ajouter sasfr.com dans Vercel → projet sasfr-hub → Settings → Domains

Ne pas modifier la zone DNS automatiquement — action manuelle réservée à l'utilisateur.
